
# Distribuidora Jireth

Diseño de una pagina web sencilla para una pequeña distribuidora de productos y abarrotes hecho con HTML, CSS Y Javascript


## 📋Requerimientos
- Navegador moderno (Chrome, Firefox, Edge)

### Librerías principales
- HTML
- Javascript
- CSS (estilos)
## ⚙️ Instalación - configuración

1. Clona el repositorio:

```bash
  git clone https://github.com/Maxin-Mendez/Distribuidora-Jireth.git
```

2. Corre el programa:
```bash
  Puedes correrlo desde el index.html o tambien desde el login.html
```

3. Los usuarios y sus respectivas contraseñas para el login.html son las siguientes:
```bash
    admin: "admin123",
    user: "mondongo",
    gerente: "gerente_waos"
```
